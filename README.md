# Smart Gym MERN Starter

This starter project is based on the Smart Gym case study requirements:
- account creation and sign in
- membership status and renewal
- QR check-in
- class browsing, booking, cancellation, and waitlist
- trainer roster and attendance
- admin class and plan management
- reports and payment flow

## Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: MongoDB + Mongoose

## Run
### Backend
```bash
cd backend
npm install
cp .env.example .env
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
